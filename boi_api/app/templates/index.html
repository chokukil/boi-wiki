보이는 범위

  • Public: 전 구성원 접근 가능
  • Team: 사번의 Team ACL 기준
  • Private: Web/Langflow가 BoI Wiki에 저장한 본인 항목
  • Local Private: 로컬 Agent에만 저장된 항목은 Web 미노출

Event Broker가 보이는 방식

Kafka는 뒤에서 이벤트를 전달하고, Wiki는 업무 사용자가 이해할 수 있도록 Event Type과 Event Stream을 보여줍니다.

  • Event Type Catalog: 어떤 업무 이벤트가 어떤 BoI/Flow를 만드는지
  • Event Stream Actions: 어떤 이벤트가 발행/처리되었는지
  • BoI 문서: source_event.event_type으로 업무 맥락과 연결

이벤트 발행 테스트

아래 명령 또는 API로 Kafka 이벤트를 발행하면 event-adapter가 처리해 Private BoI를 생성합니다.
