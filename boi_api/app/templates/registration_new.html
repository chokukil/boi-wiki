진행 순서

  1. 추천 받기자연어 설명으로 기존 Event/SOP/Action을 먼저 찾습니다.
  2. 선택/보강필요한 섹션만 선택하고 부족한 항목을 보강합니다.
  3. 초안 만들기운영 반영 전 SOP 실행 흐름 draft로 저장합니다.
  4. 검증/게시 요청권한과 필수 정보를 확인한 뒤 게시 요청합니다.

SOP 실행 흐름

처음부터 정확한 ID를 알 필요는 없습니다. 업무를 설명하면 Agent가 후보를 추천하고, 선택값을 각 섹션에 반영할 수 있습니다.

1

Event

요일
